Hexo 优化

2015-03-17 评论 Hexo

#RSS 插件
使用插件hexo-generator-feed能生成Atom 1.0或者RSS 2.0 feed。安装很简单

然后在Hexo根目录下的_config.yml里配置一下

  • type 表示类型, 是 atom 还是 rss2.
  • path 表示 Feed 路径
  • limit 最多多少篇最近文章
    根据使用的主题不同配置也可能不同，我使用的是jacman，RSS配置就在主题目录下的_config.yml文件里。
