##CRYPTonite

先搜索spanish的书，找到Don Quijote，下载后，统计词频，然后替换，代码如下：

import collections


def counter(filename):
    f = open(filename, "rb")
    text = f.read()
    count = collections.Counter(text)
    f.close()
    return count


def decode(alphabet, ciphertext, key):

    dic = {}
    for i in range(0, len(key)):
        dic[key[i]] = alphabet[i]

    plaintext = ""
    for l in ciphertext:
        if l in dic:
            l = dic[l]
        plaintext += l

    return plaintext


if __name__ == "__main__":
    # count1 = counter('spanish-book.enc')
    # count2 = counter('DON_QUIJOTE.txt')
    # print count1
    # print count2

    f = open('spanish-book.enc', "rb")
    ciphertext = f.read()
    f.close()
    print decode("EAOSNRLDIUTCM,QPBYHVGJF.Z;,!XS-W-K7", ciphertext, "_C!BI)LRUPMQNA,.:GF;-V?SZDXKEHYO(JT")

在替换后的内容中搜索NCN，搜索到NCNKDEADBEAFCAFEBADBABEFEEDDEFACEDBEDFADEDEC，将其中的K替换为_即可。

##MISCall
查看文件类型，为bz2文件，解压后得到flag.txt，但里面的内容没用，还有一个.git目录，从里面可以看到是initial commit。然后，执行git stash pop命令，得到新的flag.txt和s.py文件，s.py文件内容如下：

#!/usr/bin/env python
from hashlib import sha1
with open("flag.txt", "rb") as fd:
    print "NCN" + sha1(fd.read()).hexdigest()

直接运行s.py，得到flag：NCN4dd992213ae6b76f27d7340f0dde1222888df4d3
